# 💅 css/

## 🇬🇧 English
This directory contains all the Cascading Style Sheets (CSS) files responsible for the visual presentation and styling of the project. The styles are modularized as follows:

- **[style.css](style.css)**: The main stylesheet that imports all other style files and defines global styles.
- **[header.css](header.css)**: Styles for the header section, including the logo and navigation bar.
- **[main.css](main.css)**: Styles for the main content of the page, including the hero, dynamic, and playground sections.
- **[footer.css](footer.css)**: Styles for the footer, including social media links and copyright information.
- **[wrapper.css](wrapper.css)**: Styles for the main wrapper that contains the entire page content.
